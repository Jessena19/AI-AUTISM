# AUTIS – AI-Based Autism Interaction & Monitoring System

## 📌 Overview

AUTIS (AI-Based Autism Interaction & Monitoring System) is an AI-powered assistive platform designed to support behavioral training, engagement assessment, and interaction monitoring for children with Autism Spectrum Disorder (ASD).

The system combines Computer Vision, Audio Interaction, and Behavioral Analysis to create a structured learning environment that encourages participation while providing measurable insights for parents, therapists, and educators.

> **Disclaimer:** AUTIS is not intended to diagnose Autism Spectrum Disorder (ASD). It serves as an assistive tool for behavioral monitoring, engagement assessment, and interaction-based training.

---

## 🎯 Objectives

* Improve child engagement through structured interactions.
* Monitor behavioral responses using vision and audio-based analysis.
* Measure attention, response time, and task completion.
* Provide positive reinforcement and guided interaction.
* Enable continuous home-based support alongside professional therapy.

---

## 🚀 Key Features

### 👁️ Visual Attention Monitoring

* Face detection and tracking
* Eye gaze estimation
* Head movement analysis
* Attention level assessment
* Smile detection
* Hand-raise and gesture recognition

### 🔊 Audio Interaction System

* Pre-recorded audio instructions
* Sequential interaction flow
* Child response monitoring
* Positive reinforcement feedback
* Reminder prompts for missed responses

### 📊 Behavioral Analysis

* Response latency measurement
* Engagement scoring
* Attention tracking
* Session-wise performance evaluation
* Behavioral trend monitoring

### 🤖 AI-Based Assessment

* Real-time computer vision processing
* Adaptive interaction workflow
* Automated engagement analysis
* Structured response evaluation

## 🔊 Audio Interaction Flow

| Audio File | Purpose                   |
| ---------- | ------------------------- |
| 001.wav    | Greeting Prompt           |
| 009.wav    | Name Response Interaction |
| 0011.wav   | Raise Your Hand Activity  |
| 0013.wav   | Clap Your Hands Activity  |

The system automatically plays audio prompts in sequence and waits for the child's response before proceeding to the next interaction.
